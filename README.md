## <p align="center">AR Magic Book</p> ##  
<p align="justify">Η εφαρμογή AR Magic Book χρησιμοποιεί την τεχνολογία της επαυξημένης πραγματικότητας (Augmented Reality) στοχεύοντας στην διαδραστική διδασκαλία της ποίησης του Οδυσσέα Ελύτη μέσα στην σχολική αίθουσα. Απευθύνεται σε μαθητές της Πρωτοβάθμιας και Δευτεροβάθμιας Εκπαίδευσης. Μέσω της επαυξημένης πραγματικότητας, η διδασκαλία στην σχολική τάξη εμπλουτίζεται με εικονικά/ψηφιακά collage, ήχους και μουσικές, κάνοντας τα ποιήματα του Οδ. Ελύτη να “ζωντανεύουν” μέσα από την οθόνη των smart phones. Η διάδραση με τον κόσμο της επαυξημένης πραγματικότητας ενισχύει την εκπαιδευτική διδασκαλία, καθώς επιτρέπει τους μαθητές να εξερευνήσουν τον ποιητικό κόσμο του Ελύτη, μέσω της συν-ανάγνωσης των σουρρεαλιστικών εικόνων/collage, που προβάλλονται στην οθόνη του κινητού, με τους στίχους του ποιημάτων του. Το νέο, αναγεννημένο, συνθετικό collage που δημιουργείται με την τεχνολογία της επαυξημένης πραγματικότητας, ακροβατεί μεταξύ δύο κόσμων: του πραγματικού/υλικού και του εικονικού/ψηφιακού.</p>  

### Εργαλεία και Τεχνολογίες που χρησιμοποιήθηκαν για την σχεδίαση της εφαρμογής ###  
1)	Unity game engine, έκδοση editor 6000.0.42f1  
2)	Google Play Services for AR ver. 1.38.231530493 (arm64-v8a) (nodpi) (Android 12+)  
3)	Vuforia engine ver. 11.4.4 (plugin για Unity)   
4)	On-line εργαλείο στο Developers Portal της Vuforia για Rating των εικόνων που πρόκειται να χρησιμοποιηθούν για εφαρμογές Augmented Reality  
5)	On-line πλατφόρμα σχεδίασης canva  
6)	MS Visual Studio 2023
7)	C# για scripting
8)	GitHub για φιλοξενία του repository της εφαρμογής
9)	Smartphone Galaxy A14 SM-A145R/DSN με Android 15 (One UI 7.0), API 35, kernel 6.6.56
